# Web_course-backend
A backend system for an online learning platform, designed to deliver seamless course streaming and educational content management.

## Technologies Used
- NextJS
- MySql

## Requirements
- A modern, up-to-date web browser (e.g., Chrome, Firefox, Edge, or Safari)
